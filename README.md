# SOP PROJECT 2022
เว็บยืมหนังสือและเช่า WorkSpace โดย React.js และ Java Spring Boot

### FrontEnd Framework ที่ใช้
- React.js
#### CSS framework
- TailwindCSS
- MaterialUi
### Microservice Backend
- Spring Boot

# วิธีใช้งาน (FrontEnd)
1. Clone Project
2. เปิด Editor เช่น VSCode
3. เลือก Folder ที่ Clone ลงมา
4. cd frontend
5. npm install
6. npm start
7. รอจนมันเปิด
